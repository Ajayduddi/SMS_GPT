Understanding the Code

The project is built with Python and uses desktop automation to control the browser.

System Architecture

When a user sends an SMS, it syncs to Google Messages Web via their paired Android phone. The Python script monitors the browser, extracts new messages, sends them to Gemini AI for processing, and automatically types the response back into the chat.

main.py

The core automation engine. Launches Edge browser, navigates to Google Messages, and runs a continuous loop to detect and respond to new SMS messages.

Key: SMSBot class, process_loop()

gemini.py

Handles communication with Google's Generative AI API. Manages chat sessions, sends prompts, and returns AI-generated responses.

Key: send_message(), send_audio()

config.py

Central configuration file. Stores API keys, URLs, audio settings, and other environment variables used across the project.

Key: GEMINI_API_KEY, NEWS_API_KEY
